Overview

Khám phá Mini Finance

Mini Finance là một ứng dụng web thông minh, giúp phân tích bài viết và xây dựng bộ từ điển chuyên biệt với giao diện thân thiện và công cụ mạnh mẽ. Sử dụng Flask, PostgreSQL và Bootstrap, ứng dụng mang đến trải nghiệm mượt mà và trực quan. Các tính năng nổi bật bao gồm:

  • Tổng quan: Cung cấp hướng dẫn sử dụng và giới thiệu tổng quát về ứng dụng.
  • Dữ liệu bài viết: Quản lý danh sách bài viết với thông tin chi tiết (ID, ngày, liên kết, mô tả) và hỗ trợ phân trang.
  • Phân tích chuyên sâu: Phân tích nội dung bài viết bằng thư viện underthesea, đếm ký tự, từ, cụm từ có nghĩa, và hỗ trợ ghép từ thủ công để xây dựng từ điển.
  • Quản lý từ khóa: Lưu trữ từ khóa với loại từ và ghi chú, tối ưu hóa việc phân loại và tra cứu.
  • Giao diện hiện đại: Thiết kế responsive, tích hợp thanh điều hướng tiện lợi.

Hướng dẫn triển khai

Để chạy Mini Finance trên máy cục bộ, thực hiện các bước sau:

  1. Yêu cầu: Python 3.6+, PostgreSQL, và các thư viện flask, flask_sqlalchemy, underthesea, psycopg2.
  2. Chuẩn bị môi trường:
    • Sao chép các file app.py, index.html, articles.html, profile.html vào thư mục dự án.
    • Tạo thư mục static và templates cho CSS, JS, và HTML.
    • Cài đặt: pip install flask flask_sqlalchemy underthesea psycopg2-binary.
  3. Cấu hình PostgreSQL:
    • Đăng nhập: psql -U postgres.
    • Tạo database: CREATE DATABASE postgres;.
    • Tạo bảng: 
      CREATE TABLE wallet (
    id SERIAL PRIMARY KEY,
    date VARCHAR(50) NOT NULL,
    time VARCHAR(20) NOT NULL,
    link VARCHAR(255),
    description TEXT
);
CREATE TABLE keyword (
    id SERIAL PRIMARY KEY,
    word VARCHAR(255) UNIQUE NOT NULL,
    loai_tu VARCHAR(50),
    ghi_chu TEXT
);
  4. Cấu hình ứng dụng: Cập nhật chuỗi kết nối trong app.py (ví dụ: postgresql://postgres:220201@localhost:5432/postgres).
  5. Khởi chạy: Chạy python app.py và truy cập http://127.0.0.1:5000.
  6. Khám phá: Điều hướng đến Article Data hoặc Analysis để trải nghiệm.

Lưu ý:

  • Đảm bảo PostgreSQL đang hoạt động.
  • Thêm dữ liệu mẫu: INSERT INTO wallet (date, time, link, description) VALUES ('2025-04-17', '10:30', 'https://example.com', 'Mua sắm tại siêu thị Coopmart');.
